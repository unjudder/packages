unjudder packages
=================

Introduction
------------
This is a [composer](http://getcomposer.org "composer - package manager") repository.

How to use?
-----------
Simply add the following lines to your project's composer.json.

	"repositories": [{
    	"type": "composer",
    	"url": "https://raw.github.com/unjudder/packages/master"
    }]
